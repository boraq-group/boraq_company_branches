Company Branches

Using this module allow user to create multi brances for the same company which has multi branches with different names, logos...,etc.

Features
  • Create Multi Branches.
  • Choose Document Templates for the each branch.
  • Choose Branch for Sales and Purchse Orders.
Configuration

1- Settings -> Users&Companies -> Branches.



2- Click on Create button.



3- Fill Branch Information.



4- Choose branch document template:

Note: The first 4 templates are for the company, so you have to choose between the last 4 templates.



5- Check the created branches under company:

How to Use:

From Sale/Purchase quotation, click on create

Select the branch which the transaction is belong to

Print Quotation/ order
